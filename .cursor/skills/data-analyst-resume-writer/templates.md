# Templates

## Intake Checklist

Ask for these only when missing:

- Target role and target company type
- Total years of experience
- Current or recent job titles
- Industry domain
- Tools: SQL, Python, Excel, Tableau, Power BI, Looker, R, statistics, experimentation
- Most important projects
- Metrics improved or monitored
- Resume language: Chinese, English, or bilingual

## Bullet Rewrite Template

Use this structure when the user gives rough project notes:

```markdown
原始信息：
- [用户原始描述]

改写版本：
- 四字总结：围绕 [业务问题/分析对象]，通过 [方法/工具] 完成 [分析/系统/模型/看板]，实现 [结果]，支持 [业务影响]

英文版本：
- Four-word label: Addressed [business problem/scope] through [method/tool], delivered [analysis/system/model/dashboard], achieved [outcome], and supported [business impact].

待补充信息：
- [缺失的指标、业务规模、时间范围、角色边界]
```

## Project Formatting Rule

Use this structure when one job contains multiple projects:

```markdown
### 公司名 | 职位
[任职时间] | [城市]

#### 项目名称
[项目时间]
- 四字总结：具体说明
- 四字总结：具体说明

#### 项目名称
[项目时间]
- 四字总结：具体说明
```

If a bullet wraps onto multiple lines in HTML or rich text, keep the description aligned as a text block rather than indented under the bullet symbol. Prefer compact one-page spacing when the content volume allows.
